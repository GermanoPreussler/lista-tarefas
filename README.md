# Lista de Tarefas

Feito com Java e Spring Boot

## Funcionalidades
- Adicionar tarefa
- Remover tarefa
- Persistência de dados com H2

## Como Rodar

Requisitos - (Java 21 e Maven instalados)

### Opção 1 - Maven
```
git clone https://github.com/GermanoPreussler/lista-tarefas
```
```cd lista-tarefas```

```mvn spring-boot:run```

### Opção 2 — JAR

```git clone https://github.com/GermanoPreussler/lista-tarefas```

```cd lista-tarefas```

```mvn package -DskipTests```

```java -jar target/ListaTarefas-0.0.1-SNAPSHOT.jar```

### Acessar

Depois de rodar, acesse no seu navegador: http://localhost:8080
